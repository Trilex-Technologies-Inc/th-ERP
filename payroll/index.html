Payroll is an LAMP application for payroll processing. The purpose with the application is to provide an application which is easy to install, maintain and host. It is not aimed for usage in countries which have not too complicated laws and rules regarding taxation, etc.

The features provided by Payroll is:

  • Standard salary calculation
  • Daily salary calculation
  • Hourly salary calculation
  • Tax calculation
  • Deductions calculation
  • Absence and attendence reporting
  • Schedule definitions
